# 功能清单与说明

## 1. 扫描与索引

1. 扫描任务启动、状态查询、暂停、取消。
2. 扫描过程流式更新，记录实时可见。
3. 历史任务可回载查看。

## 2. 搜索与筛选

1. 查询输入支持关键模式。
2. 后缀包含/排除筛选。
3. 大小区间筛选。
4. 风险类型筛选。
5. 文件/目录视图切换。

## 3. 去重与风险

1. 内容级重复组（高置信）。
2. 重复组聚焦主列表。
3. 成员数与可释放空间展示。
4. 风险标记展示与筛选。

## 4. 详情与“这是什么”

1. 详情区展示路径、大小、类型、时间、状态。
2. “这是什么”同级模块展示分析结果。
3. 输出摘要、删除建议、置信度、影响、理由。
4. 置信度中文化展示（高/中/低）。

## 5. 压缩与归档

1. 模式：视频、图片、PDF、仅打包。
2. 批量：视频批量、归档批量。
3. 格式：zip/tar。
4. 预设：视频档位、图片策略、PDF 预设。

## 6. 安全控制

1. 敏感操作请求校验（来源、客户端、token）。
2. 路径边界校验（必须在授权扫描根内）。
3. 请求体大小限制，防止内存过载。

## 7. 本地化与品牌

1. JSON 词典驱动 i18n（zh/en）。
2. 产品名更新为 DiskOrg Control Panel（SCD）。
3. 文案持续统一中。
