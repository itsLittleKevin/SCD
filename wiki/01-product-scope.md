# 产品总览与范围

## 1. 产品目标

SCD 面向本地文件治理，目标是让用户在不盲删、不失控的前提下，完成以下任务：

1. 快速定位：在大量文件中搜索、筛选、排序。
2. 可解释去重：识别高置信重复内容并估算可释放空间。
3. 风险识别：对加密、不可读、占位、异常路径等进行标记。
4. 体积治理：对视频/图片/PDF/归档进行压缩或打包。
5. 决策辅助：通过“这是什么”给出删除建议与置信度。

## 2. 当前版本范围（已实现）

1. 本地扫描与实时索引展示。
2. 内容级去重（快速指纹 + 完整哈希）。
3. 风险标记、重复组导出、任务回载。
4. 压缩与打包（单项 + 批量，zip/tar）。
5. 智能分析设置与结果展示。
6. 自动清理（Retention）策略。
7. zh/en 双语（JSON 词典驱动）。

## 3. 当前不在范围

1. 自动无人值守清理与自动删除。
2. 系统级实时监听（USN/fanotify 全覆盖）。
3. 感知相似度去重（视觉/语义）。
4. 归档深层内容索引。

## 4. 关键设计原则

1. 安全优先：默认分析与建议，不默认执行破坏性操作。
2. 可解释：每项风险与建议都应有理由。
3. 可恢复：扫描结果与设置可持久化。
4. 可运营：通过清晰 UI 与稳定流程支持长期迭代。
